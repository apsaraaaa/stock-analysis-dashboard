# Stock Analysis Dashboard

## Project Overview
This project analyzes historical stock and revenue data for Tesla and GameStop, creating interactive dashboards to visualize the data.

## Features
- Extracted stock data using yfinance API
- Web scraped revenue data from company reports
- Created interactive plots using Plotly
- Comparative analysis of Tesla vs GameStop

## Technologies Used
- Python
- yfinance
- Pandas
- Plotly
- BeautifulSoup

## Files
- `stock_analysis_dashboard.ipynb` - Main Jupyter notebook with all analysis
- `README.md` - Project documentation

## How to Run
1. Clone the repository
2. Install requirements: `pip install yfinance pandas plotly beautifulsoup4 html5lib`
3. Open and run the Jupyter notebook
